# react-template-pc react 项目模板

## 项目运行

```shell
git clone git@github.com:chenjiajing23/react-template-pc.git

cd react-template-pc

yarn

npm run dev

```

## 目标功能

- [x] 支持 typescript 的写法
- [x] antd 按需加载功能
- [x] git 检测功能
- [x] 热更新功能
- [x] less 模块化支持
- [x] webpack 缓存功能
- [ ] apis 多 api 功能
- [x] mock 数据
- [ ] upload 上传到 cdn 功能
